# UnderSuperWax

UnderSuperWax is a SuperWax fork focused on one thing: reliable permanent waterproofing (not only rain) for wood builds with cool Wax look.

Your waxed pieces are protected from:

1. Rain weather damage.
2. Water submersion damage.

## What You Get In-Game

1. Craftable Beewax at the workbench.
2. Hammer misc action to apply wax directly to wood pieces.
3. Immediate apply feedback (message + effect).
4. Persistent wax state until the piece is destroyed.

## How It Works in Practice

1. Craft Beewax.
2. Select the UnderSuperWax apply tool in your hammer misc tab.
3. Left-click a valid wood or core wood building piece.
4. One Beewax is consumed.
5. The piece is now treated as covered/dry/waterproof.

Result:

1. Better-looking wood over time.
2. Reliable protection in wet and submerged build zones.
3. No reapplication loop for normal play.

## Features

1. Permanent per-piece waterproofing.
2. Rain and submersion protection.
3. Preserved Beewax visuals (model + icon).
4. Configurable shine (Glossiness/Metallic).

## Compatibility Notes

1. Requires BepInEx.
2. Requires Jotunn.
3. Applies only to eligible wooden `WearNTear` pieces.
