# UnderSuperWax

UnderSuperWax is a SuperWax fork that keeps the Beewax fantasy and visuals, while delivering reliable protection for wood builds in wet environments.

## Player-Facing Features

1. Craft Beewax at the workbench.
2. Apply it with the hammer misc action.
3. Permanently protect valid wood/core wood pieces.
4. Block both rain damage and submersion damage.

## How to Use

1. Craft Beewax.
2. Open hammer build menu and select the UnderSuperWax apply piece in Misc.
3. Left-click a valid target piece.
4. Beewax is consumed and the piece is now waterproof.

## Installation

1. Install BepInExPack for Valheim.
2. Install Jotunn.
3. Place `UnderSuperWax.dll` into `BepInEx/plugins`.
